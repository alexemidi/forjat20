# Forja T20

Criador de personagem para Tormenta20 feito com React e Vite.

## Como rodar

```powershell
npm install
npm run dev
```

Abra `http://127.0.0.1:5173` no navegador.

## Scripts

- `npm run dev`: inicia o servidor local de desenvolvimento.
- `npm run build`: gera a versão de produção em `dist/`.
- `npm run preview`: serve a versão gerada em `dist/`.
- `npm run lint:data`: valida os catálogos de dados.
